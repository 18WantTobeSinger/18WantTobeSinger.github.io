在圆内随机生成点
Created 2020-05-01| Updated 2020-07-05
Word count:323|Reading time: 1 min
|Post View:

今天在LeetCode上刷到了一道生成随机数的题目：在圆内随机生成点

这里讲解官方题解的第二种方法（极坐标方法）；本文中的radius、x_center、y_center分别是圆的半径和圆心所在的坐标点；

思路：

既然我们要生成随机点，要求这个点在这个圆内，我们不妨以圆心为出发点。

1、首先，既然我们以圆心出发，那我们首先要确定的是方向

2、方向我们确定之后，在确定我们要走的距离，不难想到，要走的距离h应满足: 0 <= h <= radius。

3、方向和距离确定之后，我们就事半功倍了，利用三角函数求出其坐标（x,y）。

好像已经可以了，方向、距离、（x，y）我们都确定了，接下来我们看看生成的随机点是否是均匀的：

咦？？？我们可以清晰地看出大多数点分布在靠近圆心周围的区域。为了解决这个问题，我们只需要将计算距离产生的随机数进行开方即可：
