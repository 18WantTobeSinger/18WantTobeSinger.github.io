位运算的一些骚操作

发表于 2020-04-28 更新于 2020-04-29

##基本的位运算

& 与： 只有当两位都是 1 时结果才是 1，否则为 0 。

| 或：两位中只要有 1 位为 1 结果就是 1，两位都为 0 则结果为 0。

^ 异或：两个位相同则为 0，不同则为 1

~ 取反： 0 则变为 1，1 则变为 0

<< 左移：向左进行移位操作，高位丢弃，低位补 0

>> 右移：向右进行移位操作，对无符号数，高位补 0，对于有符号数，高位补符号位
